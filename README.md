*****Sentiment Analysis Web App | Python, Scikit-learn, NLTK, Pandas*****


• Built a text classification pipeline to predict sentiment (positive/negative/neutral) from user reviews using NLP
techniques.
• Performed data preprocessing including tokenization, stopword removal, and TF-IDF vectorization using NLTK
and Scikit-learn.
• Trained and compared multiple ML classifiers (Naive Bayes, Logistic Regression, SVM) achieving 87%+ accu
racy on test data.
• Visualized model performance using confusion matrix, ROC curve, and feature importance plots with Matplotlib
and Seaborn.
